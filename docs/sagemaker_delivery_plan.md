# SageMaker Delivery Plan

## Week 1 - Data and Environment

- Confirm SageMaker container and Python version.
- Validate Google/Meta exports and call-tracking data.
- Standardize weekly date grain.
- Build preprocessing pipeline.
- Produce data quality report.

## Week 2 - First Working MMM

- Implement PyMC-Marketing model.
- Add adstock and saturation transformations.
- Fit first model.
- Export posterior summaries, contributions, ROI and diagnostics.
- Build first dashboard MVP.

## Week 3 - Calibration and Validation

- Tune priors.
- Add lift tests or holdout constraints if available.
- Run posterior predictive checks.
- Review residuals and contribution plausibility.
- Stress-test response curves.

## Week 4 - Dashboard and Handoff

- Finalize Streamlit views.
- Add budget recommendation logic.
- Document assumptions and limitations.
- Deliver technical handoff and executive interpretation guide.
